# Table des Matières: Guide Commercial Showroom CERASCAN

**Type:** user-guide  
**Client:** CERASCAN (EFFICERAM)  
**Version:** 1.0  
**Date:** 2026-02-28

---

## Vue d'ensemble

Guide opérationnel ultra-compact pour commerciaux showrooms équipés CERASCAN. Usage quotidien : consultation rapide tablette/smartphone pendant service clients. Fonctionnalités outil uniquement (pas conseils commerciaux). Format terrain 6-8 pages, plastifiable comptoir showroom. Lecteur formé 30 minutes par propriétaire showroom.

---

## Concepts Principaux

- **Compte commercial :** Accès privilégié application (prix professionnels, stocks temps réel, marges calculées)
- **Catalogue produits :** 50-2000 produits navigables par catégories (Intérieur/Extérieur, finitions, fournisseurs)
- **QR codes échantillons :** Scan appareil photo tablette/smartphone ouvre fiche produit instantanément
- **Listes d'envie visiteurs :** Visiteurs sauvegardent produits consultés, commerciaux accèdent historiques pour relance
- **Infos privilégiées :** Prix professionnels (vs prix public visiteurs), stocks disponibilité temps réel, marges calculées automatiquement

---

## Audience

**Vous = Commercial/Vendeuse showroom**
- Utilisez application CERASCAN quotidiennement (tablette/smartphone)
- Accompagnez visiteurs recherche produits
- Compétences : usage smartphone/tablette basique, navigation web

### Chapitre 02 : Se Connecter

**Synopsis :**  
Accéder application CERASCAN : URL fournie propriétaire showroom (ex: https://partner808.cerascan.fr) ou QR code accès rapide collé comptoir showroom (scan ouvre navigateur préconfiguré). Identifiants commercial : email + mot de passe temporaire fournis propriétaire lors formation 30 min. Premier login obligatoire changement mot de passe (sécurité, mémorisation personnelle). Page connexion : champs email/mot de passe, bouton "Se connecter", case "Rester connecté" (tablette showroom fixe, déconseillé smartphone personnel). Connexion réussie : redirection catalogue produits automatique. Se déconnecter : menu profil (icône utilisateur haut droite), bouton "Déconnexion". Changer session commercial : si plusieurs commerciaux utilisent même tablette, déconnexion obligatoire avant connexion suivant (traçabilité consultations).

**Points clés :**
- Accès application : URL https://{partenaire}.cerascan.fr (fournie email propriétaire), QR code accès rapide collé comptoir showroom (scan ouvre navigateur direct), ajouter favoris navigateur smartphone/tablette (accès 1 clic quotidien)
- Identifiants commercial : Email professionnel showroom (ex: julie@showroom-ceramique.fr), mot de passe temporaire fourni propriétaire (ex: "CERASCAN2024!"), créés par propriétaire interface partenaire section "Utilisateurs"
- Premier login sécurisé : Page "Changer mot de passe" apparaît automatiquement premier login, champs (mot de passe temporaire, nouveau mot de passe, confirmation), validation affiche message "Mot de passe changé, connexion automatique"
- Case "Rester connecté" : Tablette showroom fixe (recommandé, pas redemander identifiants chaque jour), smartphone personnel commercial (déconseillé, sécurité données), durée session 30 jours (reconnexion automatique après 30 jours inactivité)
- Déconnexion : Menu profil icône utilisateur (coin haut droite interface), bouton "Déconnexion" (confirmation), redirection page connexion, nécessaire si plusieurs commerciaux partagent tablette (traçabilité)
- Problèmes connexion courants : Mot de passe oublié (contacter propriétaire showroom, réinitialisation interface partenaire), compte bloqué après 5 tentatives échouées (déblocage propriétaire), URL application erronée (vérifier URL email propriétaire)

### Chapitre 03 : Consulter le Catalogue Produits

**Synopsis :**  
Navigation catalogue produits : page accueil affiche catégories principales (Intérieur/Extérieur, finitions Mat/Brillant/Structuré), recherche textuelle barre haut (nom produit, référence, fournisseur), filtres avancés (prix min/max, disponibilité stock, catégories multiples, fournisseurs sélectionnés). Scanner QR code produit physique showroom : icône appareil photo barre recherche, viser QR code échantillon, ouverture fiche produit automatique instantanée (<2 secondes). Fiche produit complète : photos HD (zoom pinch), dimensions (largeur × longueur × hauteur cm), finitions textuelles, catégorie (Int/Ext), prix (public si visiteur, professionnel si commercial), disponibilité stock (Disponible immédiat, Rupture temporaire, Sur commande délai X jours), produits connexes suggérés (même collection, alternatives dimensions similaires). Informations privilégiées compte commercial : prix professionnels affichés (vs prix public masqués visiteurs), stocks temps réel visibles (quantités exactes disponibles entrepôt), marges calculées automatiquement (prix achat fournisseur → prix vente public pourcentage marge). Tri résultats recherche : par pertinence (défaut), par prix croissant/décroissant, par disponibilité (disponibles d'abord), par fournisseur alphabétique. Filtres combinables : sélectionner plusieurs filtres simultanés (ex: Intérieur + Mat + Disponible + Fournisseur Imola).

**Points clés :**
- Navigation catégories : Page accueil grilles visuelles (Intérieur/Extérieur tuiles photos), sous-catégories finitions (Mat, Brillant, Structuré), arborescence 2-3 niveaux profondeur (ex: Intérieur > Mural > Céramique > Finition Mat)
- Recherche textuelle rapide : Barre recherche haut interface (icône loupe), recherche nom produit (ex: "carrelage gris"), référence exacte (ex: "INTFURN01COLL01GRAY60X60"), fournisseur (ex: "Imola"), suggestions automatiques pendant saisie (autocomplétion)
- Scanner QR code : Icône appareil photo barre recherche (ou bouton "Scanner QR" page accueil), autoriser accès caméra navigateur (popup première utilisation), viser QR code échantillon showroom (cadrage automatique), ouverture fiche produit instantanée (<2 secondes)
- Fiche produit structure : Galerie photos HD (swipe horizontal, zoom pinch-to-zoom), bloc informations (référence, nom, dimensions, finition, catégorie), prix (gros caractères), disponibilité stock (badge vert/orange/rouge), description longue (déplier "Voir plus"), produits connexes bas page (carrousel horizontal 5-10 produits)
- Infos privilégiées commerciaux : Prix professionnels (badge "Prix Pro" visible, prix public affiché grisé barré), stocks temps réel (quantité exacte "12 unités disponibles" vs visiteur "Disponible"), marges calculées (ligne "Marge: 28%" affichée sous prix pro, calcul automatique ((prix public - prix achat) / prix public) × 100)
- Filtres avancés : Panneau filtres latéral gauche (icône entonnoir), prix (sliders min/max), disponibilité (checkboxes Disponible/Rupture/Sur commande), catégories multiples (checkboxes combinables), fournisseurs (liste alphabétique checkboxes), bouton "Réinitialiser filtres" bas panneau
- Tri résultats : Menu déroulant "Trier par" haut liste résultats (Pertinence défaut, Prix croissant, Prix décroissant, Disponibilité, Fournisseur A-Z), résultats mis à jour instantanément changement tri
- Performance navigation : Cache IndexedDB (catalogue chargé local, navigation instantanée 10-50ms), synchronisation automatique mise à jour catalogue (propriétaire importe CSV, commerciaux voient nouveautés 5-10 min après), mode hors ligne limité (consultation produits déjà chargés uniquement, pas recherche serveur)

**Longueur estimée :** 2-3 pages  
**Output-style :** `user-friendly.md` (captures interface détaillées, procédures scan QR)  
**Persona :** `clarity-expert.md` (infos privilégiées explicites, filtres compréhensibles)

### Chapitre 04 : Accéder aux Listes d'Envie Visiteurs

**Synopsis :**  
Qu'est-ce qu'une liste d'envie : visiteur showroom scanne QR codes, consulte fiches produits, sauvegarde produits favoris liste personnelle, sauvegarde navigateur ou création compte (email). Commerciaux accèdent historiques listes visiteurs : menu "Listes d'Envie" (icône cœur), page liste toutes listes créées showroom (date création, nombre produits, nom visiteur si compte créé). Consulter liste visiteur : clic ligne liste ouvre détails (liste produits sauvegardés, photos miniatures, références, prix, disponibilités), historique consultations (dates/heures visites showroom, produits scannés chronologiquement). Retrouver liste visiteur spécifique : recherche par nom visiteur (si compte créé), recherche par email (si renseigné), recherche par date visite (calendrier filtre), recherche par produit référence (quels visiteurs consultés ce produit). Exporter liste : bouton "Exporter PDF" génère document imprimable (logo showroom, liste produits avec photos/prix/références, QR codes produits inclus PDF), bouton "Envoyer Email" (champ email destinataire, message personnalisable, envoi automatique PDF attaché). Statistiques produits populaires : onglet "Statistiques" page listes envie, classement produits plus sauvegardés (30 derniers jours, graphique barres), identification tendances visiteurs.

**Points clés :**
- Définition liste envie : Visiteur scanne QR code produit, clic bouton "Ajouter Favoris" fiche produit (icône cœur), produit sauvegardé liste navigateur (localStorage si pas compte), création compte optionnelle visiteur (email + mot de passe, synchronisation listes multi-appareils)
- Menu Listes Envie commercial : Icône cœur menu principal (ou onglet "Listes d'Envie"), tableau listes créées (colonnes: Date visite, Nombre produits, Nom visiteur, Actions), tri par date décroissante (récentes d'abord), pagination si >50 listes (navigation pages bas tableau)
- Détails liste visiteur : Clic ligne tableau ouvre page détails, galerie produits sauvegardés (miniatures + références + prix + stocks), chronologie consultations (dates/heures scans QR codes, parcours visiteur showroom reconstruit), durée visite showroom calculée (première consultation → dernière)
- Recherche liste spécifique : Barre recherche haut page listes envie, recherche nom visiteur (autocomplétion si comptes créés), recherche email exact, filtre calendrier date visite (sélection plage dates), recherche produit référence (affiche toutes listes contenant ce produit)
- Export PDF liste : Bouton "Exporter PDF" page détails liste, génération document A4 (logo showroom haut page, coordonnées contact showroom, tableau produits avec photos miniatures/noms/références/prix/disponibilités, QR codes produits inclus bas page réimpression possible), téléchargement automatique navigateur
- Envoi email liste : Bouton "Envoyer Email" page détails liste, formulaire (email destinataire prérempli si visiteur compte créé, champ message personnalisable "Bonjour, voici votre sélection produits...", aperçu PDF avant envoi), envoi automatique email avec PDF attaché + lien retour catalogue showroom
- Statistiques produits populaires : Onglet "Statistiques" page listes envie, graphique barres 20 produits plus sauvegardés (30 derniers jours défaut, filtre période personnalisable), pourcentages consultations (ex: "Produit INTFURN01 sauvegardé 45 fois, 12% total listes"), identification tendances saisonnières (produits populaires période)

**Longueur estimée :** 1-2 pages  
**Output-style :** `user-friendly.md` (fonctionnalités listes détaillées, captures tableaux)  
**Persona :** `clarity-expert.md` (export/email explicites)

### Chapitre 05 : Cas d'Usage Showroom

**Synopsis :**  
Cas 1 Rechercher produit précis : visiteur demande "carrelage 60×60 gris mat", commercial utilise barre recherche textuelle (saisie "gris mat 60x60"), filtres dimensions (sélection 60×60 cm), filtre finition (Mat), résultats affichés (photos produits correspondants), clic fiche produit montre détails complets. Cas 2 Comparer plusieurs produits : visiteur hésite entre 3 carrelages, commercial scanne QR codes 3 échantillons successivement (ouverture onglets navigateur multiples ou mode comparaison si fonctionnalité activée), tableau comparaison côte-à-côte (dimensions, finitions, prix, stocks, photos), visiteur choisit selon critères (prix, disponibilité immédiate). Cas 3 Consulter alternatives produit rupture stock : visiteur sélectionne produit rupture temporaire (badge orange "Sur commande 3 semaines"), commercial consulte fiche produit section "Produits Connexes" bas page, alternatives similaires suggérées automatiquement (même dimensions, finitions proches, prix équivalents, disponibles immédiatement), présentation alternatives visiteur. Cas 4 Afficher produits complémentaires : visiteur achète carrelage sol, commercial consulte section "Produits Connexes" fiche produit, suggestions automatiques (carrelage mural même collection, plinthes assorties, produits entretien), affichage suggestions visiteur (vente croisée facilitée). Cas 5 Générer aperçu liste produits : visiteur sélectionné plusieurs produits, commercial accède menu "Listes d'Envie", exporte liste PDF (logo showroom, références/prix/photos produits, QR codes réimpression), impression immédiate ou envoi email visiteur.

**Points clés :**
- Cas 1 Recherche précise : Barre recherche haut interface (saisie "gris mat 60x60" texte libre), panneau filtres latéral (checkboxes Finition Mat, slider dimensions 60×60 cm exactement), résultats filtrés instantanés (10-50 produits correspondants), tri prix croissant recommandé (visiteurs budget sensibles), clic miniature produit ouvre fiche détaillée (référence, stock, prix pro visible commercial)
- Cas 2 Comparaison produits : Scanner QR codes 3 échantillons successivement (appareil photo barre recherche, scans rapides <5 secondes total), onglets navigateur multiples ouverts automatiquement (1 onglet = 1 fiche produit), navigation swipe horizontal entre onglets (ou mode comparaison tableau si fonctionnalité activée interface), comparaison manuelle (commercial note prix/dimensions/stocks sur papier ou mémoire), présentation orale visiteur différences clés
- Cas 3 Alternatives rupture stock : Fiche produit badge orange "Rupture temporaire" ou "Sur commande délai 3 semaines" visible, scroll bas fiche section "Produits Connexes" (carrousel horizontal 5-10 alternatives), algorithme suggestions automatique (mêmes dimensions priorité, finitions similaires, même gamme prix ±10%, disponibilité immédiate priorité haute), clic alternative ouvre fiche comparaison (commercial présente "Produit similaire disponible immédiatement, dimensions identiques, finition proche")
- Cas 4 Produits complémentaires : Fiche produit sol carrelage, section "Produits Connexes" suggestions automatiques (carrelage mural même collection fournisseur, plinthes céramique assorties couleurs, produits entretien céramique spécifiques, joints/colles recommandés fournisseur), carrousel horizontal scroll (photos miniatures + noms + prix), clic produit complémentaire ouvre fiche (vente croisée naturelle, pas démarchage commercial forcé)
- Cas 5 Export liste rapide : Visiteur consulté/sauvegardé 5-10 produits session visite, commercial accède menu "Listes d'Envie" (icône cœur), identification liste visiteur (date/heure récente, nombre produits correspondant), bouton "Exporter PDF" génère document (logo showroom, coordonnées contact, tableau produits avec miniatures/références/prix/stocks, QR codes bas page), impression comptoir showroom (remise papier visiteur) ou envoi email (visiteur renseigne email, réception immédiate)

**Longueur estimée :** 2-3 pages  
**Output-style :** `user-friendly.md` (scenarios concrets détaillés, captures cas réels)  
**Persona :** `clarity-expert.md` (fonctionnalités outil explicites, pas conseils commerciaux)

### Chapitre 06 : Résolution Problèmes et Support

**Synopsis :**  
FAQ format question/réponse 7-10 questions courantes commerciaux. Q1 QR code ne scanne pas : vérifier luminosité showroom (QR code zone sombre illisible), nettoyer objectif caméra tablette/smartphone (traces doigts dégradent scan), QR code physique abîmé/déchiré (signaler propriétaire réimpression), autorisation caméra navigateur refusée (paramètres navigateur réautoriser). Q2 Connexion perdue pendant utilisation : vérifier Wi-Fi showroom (reconnexion automatique réseau), mode hors ligne activé automatiquement (consultation produits déjà chargés possible, pas recherche serveur), reconnexion automatique Wi-Fi rétabli. Q3 Informations produit manquantes ou erronées : prix manquant, stock erroné, photo absente (signaler propriétaire showroom, correction import CSV suivant), délai correction 1-7 jours selon fréquence imports propriétaire. Q4 Application lente chargement catalogue : vider cache navigateur (paramètres navigateur, effacer données navigation CERASCAN uniquement), redémarrer tablette/smartphone (fermeture applications arrière-plan), catalogue volumineux >2000 produits (temps chargement initial 30-60 secondes normal, navigation ensuite instantanée cache). Q5 Mot de passe oublié : contacter propriétaire showroom (réinitialisation interface partenaire section "Utilisateurs"), nouveau mot de passe temporaire envoyé email, reconnexion changement mot de passe obligatoire. Q6 Compte bloqué après tentatives échouées : déblocage propriétaire showroom (interface partenaire), sécurité 5 tentatives mot de passe échouées = blocage automatique 24h. Q7 Listes envie visiteurs non visibles : vérifier rôle compte commercial (propriétaire vérifie permissions interface partenaire), visiteur créé liste mode anonyme navigateur (liste sauvegardée local visiteur uniquement, pas serveur, donc invisible commerciaux). Codes erreur courants : Erreur 401 "Non autorisé" (session expirée, reconnexion nécessaire), Erreur 404 "Produit introuvable" (produit supprimé catalogue, rafraîchir page), Erreur 500 "Erreur serveur" (problème technique temporaire, réessayer 5 min, persiste contacter propriétaire). Contacts support : Propriétaire showroom (questions quotidiennes, mot passe oublié, infos produit manquantes, coordonnées affichées page connexion application), Administrateur CERASCAN via propriétaire (bugs application complexes, données erronées systématiques, email admin-cerascan@efficeram.fr, téléphone +33 X XX XX XX Lun-Ven 9h-18h). Signalement bugs/problèmes : noter détails problème (date/heure, action effectuée, message erreur exact, captures écran), transmettre propriétaire showroom (email ou verbal), propriétaire contacte administrateur CERASCAN si nécessaire escalade.

**Points clés :**
- FAQ Q1 QR code ne scanne pas : Vérifier luminosité showroom (zones sombres QR illisible, déplacer échantillon zone éclairée), nettoyer caméra tablette (chiffon doux, traces doigts fréquentes), QR code abîmé physiquement (déchiré, taché, plastification rayée → signaler propriétaire réimpression), autorisation caméra navigateur (paramètres navigateur réautoriser accès caméra CERASCAN)
- FAQ Q2 Connexion perdue : Wi-Fi showroom déconnecté (vérifier icône Wi-Fi tablette, reconnexion automatique réseau), mode hors ligne automatique (badge "Mode hors ligne" haut interface, consultation produits cache possible, recherche serveur indisponible), reconnexion automatique Wi-Fi rétabli (badge disparaît, synchronisation automatique)
- FAQ Q3 Infos produit erronées : Prix manquant/erroné, stock incorrect, photo absente/mauvaise qualité (signaler propriétaire showroom, noter référence produit exacte), propriétaire corrige import CSV suivant (fréquence import variable 1 fois/semaine à 1 fois/mois), délai correction 1-7 jours typique
- FAQ Q4 Application lente : Vider cache navigateur (paramètres navigateur, effacer données navigation, sélectionner cerascan.fr uniquement pas tous sites), redémarrer tablette/smartphone (fermer applications arrière-plan consomment RAM), catalogue volumineux >2000 produits (chargement initial 30-60 secondes normal première connexion journée, navigation ensuite instantanée 10-50ms cache IndexedDB)
- FAQ Q5 Mot passe oublié : Contacter propriétaire showroom (email/téléphone/verbal), propriétaire réinitialise interface partenaire section "Utilisateurs" ligne commercial concerné, nouveau mot de passe temporaire envoyé email commercial automatiquement, reconnexion avec temporaire + changement obligatoire

- Codes erreur courants : Erreur 401 "Non autorisé" (session expirée après 30 jours inactivité, reconnexion nécessaire), Erreur 404 "Produit introuvable" (produit supprimé/désactivé catalogue import CSV récent, rafraîchir page F5), Erreur 500 "Erreur serveur interne" (problème technique temporaire serveur, réessayer 5-10 min, persiste >30 min contacter propriétaire)
- Contact propriétaire showroom : Questions quotidiennes (mot passe oublié, compte bloqué, infos produit manquantes, application lente), coordonnées affichées page connexion application (email, téléphone, horaires disponibilité), réponse typique <24h ouvrables
- Contact administrateur CERASCAN : Via propriétaire showroom uniquement (pas contact direct commercial), bugs application complexes (dysfonctionnements répétés, données erronées systématiques), email admin-cerascan@efficeram.fr, téléphone +33 X XX XX XX Lun-Ven 9h-18h

## Annexe

### Annexe A : Gestes Tactiles Interface Mobile/Tablette

**Synopsis :**  
Référence rapide gestes tactiles CERASCAN interface mobile/tablette. Swipe horizontal (balayage doigt gauche/droite) : navigation entre onglets, scroll carrousels produits connexes, galeries photos fiches produits. Swipe vertical (balayage haut/bas) : scroll pages longues (listes produits, fiches détaillées), rafraîchir page (pull-to-refresh haut page). Tap simple (toucher court) : sélection produit, ouverture fiche, activation boutons, saisie champs texte. Tap long (toucher prolongé 1-2 secondes) : affichage menu contextuel (copier référence produit, partager lien), sélection multiple produits (si mode sélection activé). Pinch-to-zoom (pincement 2 doigts rapprochés/écartés) : zoom photos produits galeries, zoom cartes showroom (si fonctionnalité géolocalisation activée). Double-tap (2 touchers rapides successifs) : zoom rapide photos produits, retour zoom initial (zoom out). Schémas visuels gestes annotés (mains stylisées, flèches mouvements).

**Points clés :**
- Swipe horizontal : Balayage doigt gauche→droite ou droite→gauche, usages (navigation onglets multiples produits comparaison, scroll carrousels produits connexes, galeries photos fiches produits 5-10 images), vitesse balayage rapide = changement onglet, lent = scroll continu
- Swipe vertical : Balayage doigt haut→bas ou bas→haut, usages (scroll pages listes produits longues >50 résultats, scroll fiches produits détaillées descriptions longues, pull-to-refresh haut page = rafraîchir données serveur), pull-to-refresh affiche spinner chargement + message "Mise à jour catalogue..."
- Tap simple : Toucher court <0,5 seconde doigt, usages (sélection miniature produit ouvre fiche détaillée, activation boutons "Ajouter Favoris"/"Exporter PDF", ouverture filtres panneau latéral, saisie barre recherche affiche clavier virtuel), feedback visuel tap (élément surligné brièvement couleur accent)
- Tap long : Toucher prolongé 1-2 secondes doigt maintenu, usages (affichage menu contextuel options avancées "Copier référence"/"Partager lien"/"Signaler problème", sélection multiple produits mode comparaison si activé), feedback tactile vibration courte (si tablette supporte)
- Pinch-to-zoom : Pincement 2 doigts (rapprochés = zoom out, écartés = zoom in), usages (zoom photos produits galeries fiches détaillées qualité HD, zoom cartes showroom géolocalisation si fonctionnalité activée), zoom maximum 4× (qualité image préservée), double-tap alternative zoom rapide
- Double-tap : 2 touchers rapides successifs <0,5 seconde intervalle, usages (zoom rapide photos produits 2× automatique, double-tap zoomed = retour zoom initial 1×), alternative pinch-to-zoom rapide (moins précis mais plus rapide)
- Schémas visuels : Illustrations mains stylisées gestes (doigts numérotés, flèches directions mouvements, annotations texte courtes "Swipe gauche", "Pinch zoom"), légende symboles (flèche simple = mouvement lent, flèche double = mouvement rapide, cercle pointillé = zone toucher)
