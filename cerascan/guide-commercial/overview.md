# Overview: Guide Commercial Showroom CERASCAN

## Objectif du Document

Guide opérationnel destiné aux commerciaux de showrooms équipés CERASCAN pour utiliser l'application au quotidien : se connecter, consulter le catalogue produits, accéder aux informations privilégiées (prix, stocks), gérer les listes d'envie visiteurs et accompagner les clients.

## Audience Cible

**Vous = Commercial/Vendeuse showroom**
- Utilisez l'application CERASCAN quotidiennement en showroom
- Accompagnez les visiteurs dans leur recherche de produits
- Accédez à des informations privilégiées (prix professionnels, stocks)
- Consultez les listes d'envie des visiteurs
- Compétences : usage smartphone/tablette, navigation web basique

**Autres rôles mentionnés (hors périmètre ce guide) :**
- **Visiteur showroom :** Scanne QR codes, consulte fiches produits (aucune documentation nécessaire)
- **Propriétaire showroom :** Gère catalogue, import CSV, impression QR codes (voir Guide Partenaire)
- **Administrateur CERASCAN :** Gère multi-tenant, support technique (voir Guide Administrateur)

## Périmètre

### Couvert
1. **Comment Utiliser ce Guide** (0,5-1 page)
   - Structure guide (sommaire, conventions)
   - Contacts support rapides (propriétaire, administrateur)
   - Ressources complémentaires

2. **Se Connecter** (1-2 pages)
   - Accéder à l'application (URL, QR code accès rapide)
   - Identifiants commercial (fournis par propriétaire showroom)
   - Premier login : configurer mot de passe
   - Se déconnecter / changer de session

3. **Consulter le Catalogue Produits** (2-3 pages)
   - Navigation catalogue (catégories, recherche détaillée)
   - Scanner QR code produit (appareil photo tablette/smartphone)
   - Fiche produit complète (photos, dimensions, finitions, stocks, prix)
   - Informations visibles compte commercial (prix pro, stocks temps réel, marges calculées)
   - Filtres et tri (prix, disponibilité, catégories, fournisseurs)

4. **Accéder aux Listes d'Envie Visiteurs** (1-2 pages)
   - Qu'est-ce qu'une liste d'envie ?
   - Consulter listes créées par visiteurs
   - Retrouver liste d'un visiteur (recherche par nom, email)
   - Exporter liste PDF/email
   - Statistiques : produits populaires

5. **Cas d'Usage Showroom** (2-3 pages)
   - Cas 1 : Rechercher produit précis (référence, nom, catégorie)
   - Cas 2 : Comparer plusieurs produits (tableau comparaison)
   - Cas 3 : Consulter alternatives produit rupture stock (filtres disponibilité)
   - Cas 4 : Afficher produits complémentaires/connexes (suggestions automatiques)
   - Cas 5 : Générer aperçu liste produits (export PDF rapide)
   - Note : Description fonctionnalités outil uniquement, pas conseils commerciaux

6. **Résolution Problèmes et Support** (2-3 pages)
   - FAQ format question/réponse (7-10 questions courantes)
   - Codes erreur courants et solutions
   - Contacts support (propriétaire showroom, administrateur CERASCAN)
   - Procédures signalement bugs/problèmes

### Non couvert
- Import catalogue produits, génération QR codes (voir **Guide Partenaire**)
- Administration multi-tenant, gestion partenaires (voir **Guide Administrateur**)
- Architecture technique, configuration serveur
