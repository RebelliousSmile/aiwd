# Client : auto — Documentation Auto-Référentielle

## Contexte

Documentation du dépôt `generationPDF` lui-même : méthodologie, pipeline, conventions, et système de prompts. Ce client est récursif — il documente l'outil qui sert à produire les autres documentations.

**Dépôt :** `generationPDF/` (pipeline PAO Markdown → ICML → InDesign → PDF)

## Audience Cible

- **Primaire :** Utilisateurs du système (auteurs, rédacteurs techniques, équipe)
- **Secondaire :** Nouveaux arrivants sur le dépôt

## Ton & Style

- Direct et pédagogique — show, don't tell
- Exemples concrets avec vraies commandes/chemins
- Langue française, accents impeccables
- Pas de jargon sans explication à 1ère occurrence

## Contraintes

- Toujours référencer les chemins réels du dépôt
- Documenter uniquement ce qui existe (pas de spéculation)
- Maintenir la cohérence avec `CLAUDE.md` (source de vérité système)
